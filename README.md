# 🪟 Glassmorphism

A sleek UI design project built with **HTML & CSS** that demonstrates the glassmorphism design trend — featuring frosted glass-style cards, blurred backgrounds, and translucent elements.

👉 [View Live]([https://talhsheikh.github.io/glassmorphism/](https://illusionoftalha.github.io/glassmorphism/))

## 📸 Preview

The project showcases a modern landing page with:
- A glassmorphic **navbar** with smooth transparency
- A bold **hero section** with motivational content
- A styled **call-to-action** button with icon

## 🛠️ Built With

- **HTML5** — Semantic structure
- **CSS3** — Glassmorphism styling (backdrop-filter, rgba, box-shadow)

## ✨ What is Glassmorphism?

Glassmorphism is a UI design style characterized by:
- **Frosted glass** effect using `backdrop-filter: blur()`
- **Semi-transparent** backgrounds with `rgba()`
- **Subtle borders** to enhance the glass illusion
- **Layered depth** using shadows and gradients
